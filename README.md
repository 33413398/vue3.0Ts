## 项目介绍


世树七夕，是以github上 react项目为原型进行vue3（vite），ts重构与部署，并加入了数据优化处理。祝大家能对心爱的人表白成功。



### 项目使用教程：
- 初始npm i拉取依赖
- 启动项目 npm run dev
- 打包项目 npm run build

## dist包不支持本地预览

您打包项目后悔弹出应该ip地址，您点进去就能预览dist的打包后的情况（项目配置了打包预览）
然后将包放入服务器或云托管平台就可以啦
